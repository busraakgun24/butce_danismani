\# AuraAtelier



AuraAtelier, kişisel gardırop, bütçe takibi, stil temaları ve mini chatbot mantığını birleştiren Streamlit tabanlı bir uygulamadır.



\## Özellikler



\- Aylık bütçe ve kategori limiti belirleme

\- Gardıroba ürün ekleme

\- Ürün fotoğrafı yükleme

\- Kullanım başı maliyet hesaplama

\- Vintage, Gotik, Old Money ve Clean/Soft Girl tema modları

\- Gündüz/gece görünüm modu

\- Mini bütçe ve kombin chatbotu



\## Kurulum



```bash

pip install -r requirements.txt

streamlit run app.py
